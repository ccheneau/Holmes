Holmes

Light media server

Welcome to Holmes

Holmes is a Java application that implements DLNA/UPnP AV protocol and shares videos, music, pictures, podcasts (RSS) and Icecast web radio with compatible devices.

Holmes works with VLC 2.x, Play station 3, Freebox V6 (a french set top box) and LG SmartTV devices

See Holmes wiki for more information
