Welcome to Holmes.

Holmes is a Java application that implements DLNA/UPnP protocol for playing videos, music, pictures and podcasts (RSS) to compatible devices.

Holmes runs on Windows 7 and Linux Ubuntu with VLC 2.x, Freebox V6 (a french set top box) and LG SmartTV devices

See Holmes wiki for more information
